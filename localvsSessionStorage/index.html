Local and Session Storage showcase

This is a sample application that demonstrates the differences between Local and Session Storage

Local Storage

  1. Open the console in the Developer Tools of your browser
  2. Type the following to save an item in Local Storage 
     localStorage.setItem('secret1', 'supersecret1')
  3. Type the following to retrieve the item from the Local Storage 
     localStorage.getItem('secret1')
  4. Open http://localhost:3000 in a new tab and type the following. You will be able to retrieve the item from the Local Storage: 
     localStorage.getItem('secret1')
  5. Close both tabs and re-open one. Type the following and observe that the item will still be retrieved verifying that data is persisted when the browser session ends. 
     localStorage.getItem('secret1')

Sessions Storage

  1. Open the console in the Developer Tools of your browser
  2. Type the following to save an item in Session Storage 
     sessionStorage.setItem('secret2', 'supersecret2')
  3. Type the following to retrieve the item from the Session Storage 
     sessionStorage.getItem('secret2')
  4. Open http://localhost:3000 in a new tab and type the following. The item will not be retrieved verifying that Session Storage is not shared between tabs. 
     sessionStorage.getItem('secret2')
  5. Close both tabs and re-open one. Type the following and observe that the item will not be retrieved verifying that data is cleared when the browser session ends. 
     sessionStorage.getItem('secret2')
